# note_app

📝 Flutter Advanced Note App

Một ứng dụng ghi chú thông minh được phát triển bằng Flutter, hỗ trợ soạn thảo văn bản phong phú (Rich Text) và lưu trữ cục bộ an toàn.

## Tính năng nổi bật 
**Quản lý ghi chú:** Dễ dàng tạo, xem, chỉnh sửa và xóa ghi chú.
<img width="349" height="464" alt="image" src="https://github.com/user-attachments/assets/4126ca4e-2ab4-4ff1-9edd-7e52bad1587c" />

* **Rich Text Editor:** Hỗ trợ định dạng văn bản (in đậm, in nghiêng, gạch chân...) nhờ tích hợp `flutter_quill`.

* **Đính kèm hình ảnh:** Cho phép chèn hình ảnh trực tiếp từ thư viện máy ảnh vào nội dung ghi chú.
* **Lưu trữ Offline:** Dữ liệu được lưu trữ an toàn và bảo mật trên thiết bị thông qua `sqflite`. 
* **Quản lý trạng thái:** Tối ưu hóa hiệu suất hiển thị với `provider`.
* **Cảnh báo thông minh:** Tự động hiện hộp thoại cảnh báo (Discard/Cancel/Save) nếu người dùng vô tình thoát khi đang viết dở mà chưa lưu.
